- React
- Typescript
- Tailwind css

- Nestjs
- Typescript
- Postgresql

### Devops
- Nginx
- Docker
- Docker compose

## Installation

To start run, add firebase credentials. Use the env.example file to know which variable is needed from firebase.

```bash
yarn start
```
